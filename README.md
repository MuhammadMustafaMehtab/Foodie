# 🍕 Foodie — Restaurant Landing Page

A responsive, single-page restaurant/food-ordering landing page built during a **Web Designing Competition at Aptech** (Frontend Development track).

![Foodie Preview](./images/Landing%20page-preview.png)

## 📌 About the Project

**Foodie** is a modern restaurant landing page designed to showcase a food ordering brand — highlighting delicious menu items, services, and easy contact options for customers. This project was created as a competition entry to demonstrate frontend design and layout skills using core web technologies.

## ✨ Features

- Responsive Navbar with Home, Menu, Offers & Contact links
- Hero section with call-to-action buttons (Order Now / View Menu)
- Menu section with 6 featured dishes (badges: New, Discounted, Special)
- Ratings & pricing displayed for each menu item
- Services section (Free Shipping, Secure Payment, Expert Support)
- About section with experience badge (10+ Years)
- Contact section with Get In Touch details, embedded map & message form
- Footer with quick links, newsletter signup & social icons
- Fully responsive layout using Bootstrap grid system

## 🛠️ Built With

- **HTML5**
- **CSS3**
- **Bootstrap** (utility & layout classes only — no JS components used)

## 🏆 Background

This landing page was designed and coded as part of a **Frontend Web Designing Competition** during my studies at **Aptech**, where I practiced building pixel-close, responsive UI layouts using HTML, CSS, and Bootstrap.
